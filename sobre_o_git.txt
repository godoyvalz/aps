Comandos criados, o comando git add adiciona uma alteração no diretório ativo à área. Ele diz ao Git que você quer incluir atualizações a um arquivo específico no próximo commit. Git commit leva as mudanças de um ambiente local para o repositório no git, permitindo ainda a inserção de uma mensagem descritiva. O comando git push permite que você envie (ou em tradução literal, empurre) os commits de sua branch e repositório Git local para o seu repositório remoto.
